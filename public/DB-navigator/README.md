## Export Instructions

1. Export images from Figma at 2x resolution for retina displays
2. Use PNG format for screenshots and mockups
3. Use JPG format for photos (research images)
4. Optimize images before adding them to keep file sizes reasonable
5. Intervention user flow images should be 320px wide to maintain consistent layout
